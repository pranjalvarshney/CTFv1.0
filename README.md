# CTFv1.0 
- A CTF(Capture the Flag) platform for hosting competitions online which involves many hacking concepts for beginners.
- There are a total of eight levels divided into two zones.

### Key Features
1. Dashboard
2. ScoreBoard
3. Profile

## How to install
1. Install nodejs and mongoDb
2. Clone the project
3. RUN "npm install"
4. RUN "nodemon app"
5. Ready to go
